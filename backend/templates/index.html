TCP Bench

在你的 VPS 上跑一下，看看它到全球主流网站的 TCP 握手延迟怎么样。不用告诉我任何账号密码——脚本在你自己的机器上跑，结果由你自己决定要不要上传。

curl -sL {{ site_url }}/run.sh | bash
  • 测试内容：对约 60 个主流站点做 TCP 端口 443 握手延迟测试，每个站点采样 60 次
  • 脚本是纯 bash，不需要在 VPS 上装 Python，跑完自动把结果上传并给你一个报告链接
  • 上传前会把 IP 后两段打码（如 1.2.*.*），不会存你的完整 IP
  • 脚本内容公开，跑之前可以先看一眼源码再决定要不要执行
