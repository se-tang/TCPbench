TCP Bench

在你的 VPS 上运行一条命令，测量到全球主流站点的 TCP 握手延迟，快速判断这条线路的出海质量。

curl -sL {{ site_url }}/run.sh | bash
  • 测试范围：覆盖约 60 个全球主流站点，对 443 端口进行 TCP 握手延迟测试，每站点采样 60 次，兼顾速度与结果稳定性
  • 零依赖：纯 Bash 实现，基于 /dev/tcp 完成计时，无需在 VPS 上安装 Python 或任何第三方工具
  • 隐私保护：上传前自动对 IP 后两段做打码处理（如 1.2.*.*），服务端不保存完整 IP
  • 透明可信：脚本源码完全公开，建议执行前先查看内容，不放心可以逐行核对再运行
